Playo App Summary

What is Playo?

Playo is a sports ground booking platform that connects players with available sports grounds for seamless time-slot reservations. It consists of three integrated applications:

🏁

Player App

Mobile app for end users to browse grounds, check real-time availability, book slots, pay online, and check in via QR code.

🏢

Owner App

Mobile dashboard for ground owners to manage listings, set pricing, view bookings, and track earnings.

🌐

Admin Panel

Web control center for platform oversight - revenue monitoring, user management, vendor verification, catalog approval, commission config, and analytics.

✓ Location-based discovery ✓ Real-time slot booking ✓ In-app payments (UPI/Cards) ✓ QR code check-in ✓ Multi-role access control ✓ Push notifications ✓ Reviews & ratings

Technology

The stack behind every plan

All three Playo tiers share this foundation. Higher tiers add optional components like Redis, S3/MinIO, and maps SDKs.

⚙️

Backend

Bun runtime + Fastify framework (TypeScript). REST API architecture with JWT authentication and role-based middleware.

🗃️

Database

PostgreSQL 16 with Prisma ORM. Migrations, type-safe queries, and relation mapping for users, grounds, bookings, and payments.

📱

Mobile App

Flutter (Dart) for Android & iOS from a single codebase. Riverpod for state management. Razorpay SDK for in-app payments.

🌐

Admin Web (Pro+)

React / Next.js responsive dashboard for vendor verification, catalog approval, commission config, and platform analytics.

💳

Payments

Razorpay SDK integration. Starter: UPI QR only. Pro: UPI via Razorpay. Enterprise: UPI + Cards + Net Banking.

🛰

Infrastructure

Docker containerized backend. Cloud deployment (AWS/GCP). Enterprise adds S3/MinIO asset storage and Redis caching.

🔔

Notifications

Firebase Cloud Messaging (FCM) for push notifications. Enterprise only. Starter and Pro rely on in-app status updates.

🗺

Maps

Google Maps / Mapbox integration. Enterprise only includes full map view with priced pins. Starter/Pro show addresses only.
